# CTR-0
Cheaper Than aRduino is a custom devboard meant to be a cost-effective alternative to the Arduino Nano series of devboard hobbyist microcontrollers. It is based of the same RP2040 microcontroller and provides the similar if not the same performance its far more costly equivalents.
